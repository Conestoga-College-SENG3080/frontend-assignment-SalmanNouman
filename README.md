# Creddit

A React-based frontend application for browsing and favoriting posts from the Creddit Web API. This project demonstrates proficiency in React, consuming authenticated Web APIs, and using the Web Storage API for state persistence.

## Features

- **Automatic Authentication**: Uses pre-configured credentials to securely connect to the Creddit API.
- **Forum Exploration**: Search for any forum or choose from popular ones to see the top 10 "hot" posts.
- **Favorites Management**: You can save your favorite posts to local storage and view them anytime. Works even after a page reload.
- **Responsive Design**: Sort of inspired by vintage reddit but I kept it simple.

## Technology Stack

- **Framework**: React 19 (TypeScript)
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4
- **API**: Creddit Web API (Authenticated via JWT)

## Development

- `npm install`: Install dependencies
- `npm run dev`: Start development server
- `npm run build`: Build for production
